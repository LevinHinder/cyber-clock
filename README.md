# Cyber Clock

This is some additional code from Huy Vectors Smart Cyber Clock project (https://www.huyvector.org/smart-cyber-clockk)

To use this code, go into the folder "2.4" if you have the 2.4 inch screen or into the folder "1.8" and follow the instructions there.

Changes include:
1. Support for the 2.4 inch display
1. Removal of blinking of LED every second
1. Removal of Space Invader
1. Redesign of the main screen, including a history graph of the sensor data
1. Improved Pomodoro settings
1. New settings page including
    1. Change LED brightness
    1. Change speaker loudness
    1. Change time range of history graph
    1. Automatic setup of Wifi
1. General performance and UI improvements

Feel free to contribute to this project.
